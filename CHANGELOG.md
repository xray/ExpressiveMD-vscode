# Changelog

All notable changes to the ExpressiveMD extension will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-01-02

### Added

- Initial release
- Syntax highlighting for `.emd` files
- Live preview panel with auto-refresh
- YAML frontmatter support
- Control structure highlighting (`${if}`, `${for}`, `${end}`)
- Variable interpolation highlighting
- Props file selection for testing templates with data
- Keyboard shortcuts for preview commands
